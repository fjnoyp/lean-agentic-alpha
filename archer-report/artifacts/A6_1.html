Optimal Weight Calculation: Sharpe-Maximizing Framework

Step 1: Baseline Portfolio Metrics

The S&P 500 baseline (100% allocation) offers: - Return: 7.0% - Volatility: 15.0% - Excess Return (vs. rf = 3.5%): 3.5% - Sharpe Ratio: 3.5% / 15% = 0.2333

Step 2: Candidate Weight Analysis

For a Sharpe-maximizing portfolio, we test whether adding ACHR improves the risk-adjusted return. Given the negative expected return of −44.3%, this is a low-probability scenario. However, we compute the portfolio impact at illustrative weights:

Scenario A: Zero Weight (100% Baseline)

  • Portfolio Return: 7.0%
  • Portfolio Volatility: 15.0%
  • Sharpe Ratio: 0.2333
  • Status: Baseline benchmark.

Scenario B: 1% ACHR / 99% Baseline (RECOMMENDED MAXIMUM)

Portfolio Return: $$E[R_p] = w_{ACHR} × E[R_{ACHR}] + w_{Base} × E[R_{Base}]$$ $$E[R_p] = 0.01 × (−44.3\%) + 0.99 × 7.0\% = −0.443\% + 6.93\% = 6.487\%$$

Portfolio Volatility: $$σ_p^2 = w_{ACHR}^2 × σ_{ACHR}^2 + w_{Base}^2 × σ_{Base}^2 + 2 × w_{ACHR} × w_{Base} × ρ × σ_{ACHR} × σ_{Base}$$ $$σ_p^2 = (0.01)^2 × (0.60)^2 + (0.99)^2 × (0.15)^2 + 2 × 0.01 × 0.99 × 0.50 × 0.60 × 0.15$$ $$σ_p^2 = 0.000036 + 0.022052 + 0.00891 = 0.030998$$ $$σ_p = \sqrt{0.030998} = 0.1761 = 17.61\%$$

Sharpe Ratio: $$Sharpe_p = \frac{E[R_p] − r_f}{σ_p} = \frac{6.487\% − 3.5\%}{17.61\%} = \frac{2.987\%}{17.61\%} = 0.1698$$

Sharpe Deterioration: $$ΔSharpe = 0.1698 − 0.2333 = −0.0635 \text{ (−27.2\% decline)}$$

Scenario C: 2% ACHR / 98% Baseline (ILLUSTRATIVE: NOT RECOMMENDED)

Portfolio Return: $$E[R_p] = 0.02 × (−44.3\%) + 0.98 × 7.0\% = −0.886\% + 6.86\% = 5.974\%$$

Portfolio Volatility: $$σ_p^2 = (0.02)^2 × (0.60)^2 + (0.98)^2 × (0.15)^2 + 2 × 0.02 × 0.98 × 0.50 × 0.60 × 0.15$$ $$σ_p^2 = 0.000144 + 0.021609 + 0.01764 = 0.039393$$ $$σ_p = \sqrt{0.039393} = 0.1985 = 19.85\%$$

Sharpe Ratio: $$Sharpe_p = \frac{5.974\% − 3.5\%}{19.85\%} = \frac{2.474\%}{19.85\%} = 0.1247$$

Sharpe Deterioration: $$ΔSharpe = 0.1247 − 0.2333 = −0.1086 \text{ (−46.5\% decline)}$$

Step 3: Risk-Parity Comparison (Alternative Framework)

Under a risk-parity allocation (each asset contributes equally to portfolio volatility):

$$w_{ACHR} × σ_{ACHR} = w_{Base} × σ_{Base}$$ $$w_{ACHR} × 60\% = (1 − w_{ACHR}) × 15\%$$ $$w_{ACHR} = \frac{15\%}{75\%} = 0.20 \text{ or } 20\%$$

Risk-parity weight: 20% ACHR / 80% Baseline

Portfolio Return: $$E[R_p] = 0.20 × (−44.3\%) + 0.80 × 7.0\% = −8.86\% + 5.6\% = −3.26\%$$

Portfolio Volatility: $$σ_p = \sqrt{(0.20 × 60\%)^2 + (0.80 × 15\%)^2 + 2 × 0.20 × 0.80 × 0.50 × 60\% × 15\%}$$ $$σ_p = \sqrt{(12\%)^2 + (12\%)^2 + 2 × 0.12 × 0.12} = \sqrt{0.0144 + 0.0144 + 0.003456} = 17.33\%$$

Sharpe Ratio: $$Sharpe_p = \frac{−3.26\% − 3.5\%}{17.33\%} = \frac{−6.76\%}{17.33\%} = −0.390$$

Assessment: Risk-parity allocation is unacceptable — portfolio return falls below the risk-free rate, resulting in deeply negative Sharpe ratio. Not recommended.

Recommended Weight: 0–1% (Minimal Allocation)

Rationale

  1. Negative Expected Return Dominates: ACHR's 12-month expected return of −44.3% reflects a 60% probability of base/bear case outcomes (−49.7% to −81.8%), with only a 15% probability of bull case (+33.1%) that would deliver positive returns. The probability-weighted outcome is deeply negative.

  2. Sharpe Ratio Deterioration: Even a 1% allocation reduces portfolio Sharpe by 27%, equivalent to a ~270 bp return drag on a 15% volatility baseline. For every 1% added to ACHR, the portfolio loses approximately 0.0635 units of Sharpe ratio.

  3. Volatility Amplification: ACHR's 60% annualized volatility is 4× the baseline portfolio's 15%, and correlation of 0.50 with the S&P 500 is moderate but meaningful. Adding 1% ACHR increases portfolio volatility by ~260 bp (from 15.0% to 17.61%), an 17% increase in risk for negative return.

  4. Macro Sensitivity Risk: A5.2 stress tests show ACHR is highly sensitive to labor market shocks (base case wage inflation shock drives stock from $3.25 to $1.13, a −65% loss). This idiosyncratic risk is not compensated by expected return.

  5. Bull Case Optionality Has Low Probability: While the bull case offers +33% return, it requires simultaneous execution on three forward drivers (Abu Dhabi Q3 2026 launch, FAA Type Cert Q4 2026, battery 400 Wh/kg confirmation) with estimated combined ~39% all-hit probability. Betting 1%+ of capital on a 15% probability event is inefficient.

Rebalancing Rationale

Why Minimal Weight (vs. Zero)?

While a 0% weight is theoretically optimal for a current Sharpe-maximizing portfolio, a 0.5–1.0% tactical threshold may be justified if:

  1. Portfolio mandate includes thematic/ESG exposure to aerospace/eVTOL sector per A2.3 forward drivers (commercial aviation decarbonization).
  2. Bull case represents an embedded option: The 15% probability of +33% return could appeal to investors with long time horizons and conviction on FAA Type Certification acceleration and Abu Dhabi commercial launch.
  3. Diversification vs. concentration risk: A sub-1% weight introduces minimal drag while providing upside capture if execution drivers align.

When to Increase Weight Above 1%?

Increase ACHR allocation only if: - Abu Dhabi launch confirmed and operationalized (Q3 2026 milestone achieved) - FAA Type Certification delivered on schedule (Q4 2026, vs. bull case +/− 6-month window) - Production ramp validated at 150+ aircraft/year (moves toward base case execution) - Capital raise terms improve (Stellantis partnership confirmed non-dilutive)
