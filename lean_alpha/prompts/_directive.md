# Deep Research Directive (Lean Agentic Alpha v1.0)

Applies to: A2, A3, A4 specialists. Honour these rules whenever invoking
external research (web search, web fetch).

1. Take time and use full context depth; do not summarize prematurely.
2. Triangulate across **at least 3 primary public sources** for any numeric claim.
3. Label sections of your reasoning clearly:
   - **Data Extraction**
   - **Correlation Evidence**
   - **Synthesis and Implications**
4. Include counter-evidence and note data confidence levels.
5. Quantify wherever possible (rates, margins, CAGR, valuation multiples).
6. Expand reasoning by one additional step if uncertainty remains.
7. Return fully cited numeric outputs suitable for audit. **Every numeric claim
   in the artifact must have an inline citation. Anything without a verifiable
   citation is labelled `ASSUMED`.**
